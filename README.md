# Project-S.E
# 🎮 Soldier Shooter Game

A browser-based 2D action shooter game built using **HTML, CSS, and JavaScript**.  
The player controls a soldier who survives waves of enemies while collecting power-ups and improving abilities.

---

## 🚀 Features

- 🎯 Mouse-based directional shooting
- ⌨️ WASD movement controls
- 🤖 Enemies spawn in waves from the top
- 🧠 Enemy AI (tracks player movement)
- 💥 Explosion and muzzle flash effects
- ⚡ Bullet effects change every 5 waves
- ❤️ Dynamic HP bar with UI
- 🎮 Smooth animations and gameplay
- 🧪 Power-up system (after wave 2)
- 🌄 Custom background and sprites

---

🕹️ Controls

| Action        | Key / Input        |
|--------------|------------------|
| Move Up      | W                |
| Move Down    | S                |
| Move Left    | A                |
| Move Right   | D                |
| Shoot        | Mouse Click      |
| Aim          | Mouse Movement   |
